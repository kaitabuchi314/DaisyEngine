# DaisyEngine
DaisyEngine is a 2D game engine, for both programmers learning c++ and pro game developers to use.

To setup the project, clone the repo using `git clone https://github.com/kaitabuchi314/DaisyEngine.git`.
Next, you can find the folder it cloned to, and double click on DaisyEngine.sln. In the future, you will be able to use a custom build system on Mac or linux.
